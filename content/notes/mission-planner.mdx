## What it is

A Windows ground control station for ArduPilot, used for mission planning,
parameter tuning, and live telemetry during flight.

## Useful panels

- **Flight Data** - live telemetry, artificial horizon, message log
- **Config/Tuning > Full Parameter List** - direct access to every ArduPilot
  parameter, searchable by name
- **Flight Plan** - waypoint mission editor with a map overlay
